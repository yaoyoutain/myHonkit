下载honkit

npm install honkit

// 初始化 会自动创建summary(文件左侧菜单结构), readme(首页文件)文件
//可以根据summary 自动创建文件夹和文件
npm honkit init 

// 生成_book文件
npx honkit build

// 预览网站
npx honkit serve

// 打包成pdf
npx honkit pdf

简化命令

在package.json 文件中增加

{
  "scripts": {
    "build": "npx honkit build",
    "serve": "npx honkit serve",
    "book": "npx honkit init",
    "pdf": "npx honkit pdf"
  }
}
